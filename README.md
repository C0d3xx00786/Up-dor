Приложение, разработанное на C# с использованием Windows Forms, предназначено для автоматизации учёта лекарственных средств, контроля сроков годности, ведения истории продаж, управления жалобами и оформления продаж.
В этом проекте используются также дополнительные технологии, такие как:

•	База данных – Microsoft SQL Server LocalDB (файл .mdf).

•	Для работы с БД используется Dapper – лёгкий ORM, позволяющий выполнять SQL-запросы напрямую.
